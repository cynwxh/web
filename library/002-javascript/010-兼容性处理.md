### 回调 递归 闭包

回调函数，将一个函数作为参数传给另一个函数

递归函数，在函数内部调用自身

闭包函数，函数外部直接访问函数内部声明的数据
```
function fun(){
    var i=10;
    return function(){
        i
    }
}
fun()();
```
